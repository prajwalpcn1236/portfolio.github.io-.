🔗 1. PageRank Algorithm

✅ Why Used:

To rank products based on their importance and authority within the marketplace.

Analyzes the link structure between products, reviews, and user behavior to determine relevance.

⏱️ Time Complexity:

Per iteration: O(n + m) where n = nodes, m = edges

Convergence: O(k * (n + m)) where k = iterations

Typical k ≈ 50-100 iterations

🧠 Space Complexity:

O(n + m) for storing the graph

PageRank vector: O(n)

Implementation Code 

//PageRank algorithm using power iteration

#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const double DAMPING = 0.85;
const int MAX_ITER = 100;
const double EPSILON = 1e-6;

int main() {
    // Number of pages
    int n = 4;

    // Adjacency matrix (row i links to column j)
    vector<vector<int>> links = {
        {0, 1, 1, 0},  // Page 0 links to Page 1 and 2
        {0, 0, 1, 0},  // Page 1 links to Page 2
        {1, 0, 0, 1},  // Page 2 links to Page 0 and 3
        {0, 0, 1, 0}   // Page 3 links to Page 2
    };

    vector<double> rank(n, 1.0 / n);
    vector<double> newRank(n, 0.0);

    for (int iter = 0; iter < MAX_ITER; iter++) {
        for (int i = 0; i < n; i++) {
            newRank[i] = (1 - DAMPING) / n;
        }

        for (int i = 0; i < n; i++) {
            int outlinks = 0;
            for (int j = 0; j < n; j++) {
                outlinks += links[i][j];
            }
            if (outlinks == 0) continue;

            for (int j = 0; j < n; j++) {
                if (links[i][j]) {
                    newRank[j] += DAMPING * rank[i] / outlinks;
                }
            }
        }

        // Check convergence
        double diff = 0.0;
        for (int i = 0; i < n; i++) {
            diff += fabs(newRank[i] - rank[i]);
            rank[i] = newRank[i];
        }

        if (diff < EPSILON) break;
    }

    cout << "PageRank Scores:" << endl;
    for (int i = 0; i < n; i++) {
        cout << "Page " << i << ": " << rank[i] << endl;
    }

    return 0;
}

🎯 2. Learning-to-Rank (LTR)

✅ Why Used:

To optimize search result rankings using machine learning techniques.

Learns from user interactions, clicks, and purchases to improve ranking quality.

Minimal example of LTR concept using pairwise ranking is shown.

⏱️ Time Complexity:

Training: O(n * m * log m) where n = queries, m = documents per query

Prediction: O(m * f) where f = number of features

Feature extraction: O(f)

🧠 Space Complexity:

Training data: O(n * m * f)

Model parameters: O(f) for linear models

Implementation Code 

//Learning-to-Rank example using pairwise ranking

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

struct Item {
    int id;
    double score;    // Predicted relevance score
    int label;       // Ground truth relevance label
};

// Comparator for pairwise ranking
bool pairwiseCompare(const Item &a, const Item &b) {
    if (a.label == b.label) {
        return a.score > b.score;  // Same relevance: higher predicted score first
    }
    return a.label > b.label;      // Higher relevance label first
}

int main() {
    vector<Item> items = {
        {1, 0.8, 2},
        {2, 0.7, 3},
        {3, 0.9, 1},
        {4, 0.6, 2}
    };

    // Sort items by pairwise comparison
    sort(items.begin(), items.end(), pairwiseCompare);

    cout << "Ranked items (by predicted score & relevance):\n";
    for (auto &item : items) {
        cout << "Item " << item.id << ": score=" << item.score << ", label=" << item.label << endl;
    }

    return 0;
}

🌲 3. Gradient Boosted Decision Trees (GBDT)

What is GBDT?

GBDT is an ensemble machine learning algorithm that builds models in a stage-wise fashion by combining many weak decision trees. Each tree corrects the errors (residuals) of the previous ensemble. It uses gradient descent to minimize a loss function by fitting new trees to the gradients of the loss. Widely used for regression, classification, and ranking tasks because of its accuracy and flexibility.

✅ Why Used:

To create powerful ensemble models for ranking and recommendation.

Combines multiple weak learners to create a strong predictive model for product relevance.

⏱️ Time Complexity:

Training: O(T * n * m * log n) where T = trees, n = samples, m = features

Prediction: O(T * log n)

Feature importance: O(T * n)

🧠 Space Complexity:

Model storage: O(T * nodes_per_tree)

Training: O(n * m) for data

Implementation Code 

// Simplified conceptual GBDT example

#include <iostream>
#include <vector>

using namespace std;

struct Tree {
    // Dummy tree model predicting a constant value
    double prediction;
    Tree(double p) : prediction(p) {}

    double predict(double x) {
        return prediction;
    }
};

int main() {
    vector<Tree> trees;
    vector<double> residuals = {1.0, 2.0, 3.0}; // Dummy residuals

    // Simulate boosting with 3 trees
    for (int i = 0; i < 3; ++i) {
        double pred = 0.5 * (i + 1); // Dummy prediction
        trees.emplace_back(pred);

        cout << "Tree " << i+1 << " predicts: " << pred << endl;

        // Update residuals (dummy update)
        for (auto&r : residuals) {
            r -= pred * 0.1; 
        }
    }

    // Final prediction for input x = 5 (sum of tree preds)
    double x = 5.0;
    double final_pred = 0.0;
    for (auto& tree : trees) {
        final_pred += tree.predict(x);
    }

    cout << "Final GBDT prediction: " << final_pred << endl;

    return 0;
}
