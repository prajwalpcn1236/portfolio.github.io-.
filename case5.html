🎬 1. Content-Based Filtering

✅ Why Used:

To recommend content based on movie/show features like genre, director, actors, and plot.

Provides personalized recommendations by analyzing content attributes and user preferences.

⏱️ Time Complexity:

Feature extraction: O(n * f) where n = items, f = features

Similarity computation: O(n²) for all pairs

Recommendation generation: O(n * k) where k = recommendations

🧠 Space Complexity:

Feature matrix: O(n * f)

Similarity matrix: O(n²)

Implementation Code 

//Cosine Similarity
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

double cosineSimilarity(const vector<double>& A, const vector<double>& B) {
    double dotProduct = 0.0, normA = 0.0, normB = 0.0;
    for (size_t i = 0; i < A.size(); ++i) {
        dotProduct += A[i] * B[i];
        normA += A[i] * A[i];
        normB += B[i] * B[i];
    }
    if (normA == 0 || normB == 0) return 0.0; // avoid division by zero
    return dotProduct / (sqrt(normA) * sqrt(normB));
}

int main() {
    vector<double> vec1 = {1, 2, 3};
    vector<double> vec2 = {4, 5, 6};

    double similarity = cosineSimilarity(vec1, vec2);
    cout << "Cosine Similarity: " << similarity << endl;

    return 0;
}

🧠 2. Deep Learning Models

✅ Why Used:

To capture complex patterns in user behavior and content consumption.

Neural networks can model non-linear relationships between users, content, and contextual factors.

⏱️ Time Complexity:

Training: O(E * B * L) where E = epochs, B = batch size, L = layers

Inference: O(L * W) where W = weights per layer

Backpropagation: O(L * W)

🧠 Space Complexity:

Model parameters: O(L * W)

Gradients: O(L * W)

🌐 3. Graph-Based Methods

✅ Why Used:

To model relationships between users, content, and viewing patterns as a graph.

Enables discovery of hidden connections and community-based recommendations.

⏱️ Time Complexity:

Graph construction: O(V + E) where V = vertices, E = edges

Random walk: O(W * L) where W = walks, L = walk length

Embedding generation: O(V * D) where D = dimensions

🧠 Space Complexity:

Graph storage: O(V + E)

Embeddings: O(V * D)

Implementation Code 

#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

// Example: build adjacency list for videos based on cosine similarity threshold
struct Edge {
    int to;
    double weight;
};

double cosineSimilarity(const vector<double>& A, const vector<double>& B) {
    double dot = 0, normA = 0, normB = 0;
    for (size_t i = 0; i < A.size(); ++i) {
        dot += A[i] * B[i];
        normA += A[i] * A[i];
        normB += B[i] * B[i];
    }
    if (normA == 0 || normB == 0) return 0;
    return dot / (sqrt(normA) * sqrt(normB));
}

int main() {
    vector<vector<double>> videoFeatures = {
        {1, 0, 1},
        {0.9, 0.1, 0.8},
        {0, 1, 0},
        {0.1, 0.9, 0.2}
    };

    const double threshold = 0.7;
    vector<vector<Edge>> graph(videoFeatures.size());

    for (int i = 0; i < (int)videoFeatures.size(); ++i) {
        for (int j = i + 1; j < (int)videoFeatures.size(); ++j) {
            double sim = cosineSimilarity(videoFeatures[i], videoFeatures[j]);
            if (sim > threshold) {
                graph[i].push_back({j, sim});
                graph[j].push_back({i, sim});
            }
        }
    }

    // Print graph edges
    for (int i = 0; i < (int)graph.size(); ++i) {
        cout << "Video " << i << " connected to:\n";
        for (const auto& edge : graph[i]) {
            cout << "  Video " << edge.to << " with weight " << edge.weight << endl;
        }
    }

    return 0;
}
